Welcome to SysPAC

Welcome to SysPAC - the Systems, Architecture, and Parallel Computing Group at IIITs. We are dedicated to conducting cutting-edge research and experiments across various systems, architectures, and computing software, with a focus on the intersection of system software and computer architecture. Additionally, we are committed to educating and training the next generation of researchers and professionals in Systems, Architecture, and Parallel Computing through mentorship programs, workshops, and hands-on research experiences. We will also organize a programming language and system software conference, inviting individuals from all backgrounds to share their unique projects and insights within these domains.

Current Research Projects

Register Based Custom VM

All the VM that the programming language uses are stack-based. These VM do not live up-to its performance potential as most of the time is spent on pushing and popping the constant and variable stack, so to counter this we would design a register based VM that follows CPU design. Similar routes have been followed by Lua 5.0, and have resulted in massive speed-ups.

RISC-V Instruction Emulator

Being able to inspect the source code of RISC-V is a great advantage to improve the RISC-V architecture overall. But in the process of improving its architecture there is a need to create better instruction which decodes the instruction in a better way or add new instructions to add more capabilities to its ISA. To test all these features either need a RISC-V based cpu, which is a costlier solution as each and every time you modify an instruction and you want to test it, you have to fabricate the new process for testing. We aim to design a RISC-V based Emulator for the education community to use and understand the RISCV.

Principal Investigator: Arka Mondal (arkamondalofficial@gmail.com)

WebAssembly (WASM)

WebAssembly was designed to bring the web closer to metal in terms of performance and efficiency. There are new features that are currently being proposed in the specification such as SIMD and multi-threading. And with this, it would be possible to conduct CPU intensive workloads on the web that was not previously possible. WASM as a target, will give us a way to write safe and performant code for the web easily.

Research Lead: URJASVI KUMAR SUTHAR

Secure Communication Using Quantum Computing

This project is focused on developing a comprehensive framework for secure communication by leveraging quantum computing technologies. By utilizing the distinct properties of quantum mechanics, the framework will ensure the privacy and integrity of data. It will integrate quantum key distribution (QKD) protocols, quantum encryption methods, and error correction techniques to protect communications within both classical and quantum networks. The primary objective is to create algorithms that seamlessly incorporate quantum cryptography into existing communication infrastructures, ensuring high scalability, minimal latency, and optimal security. The ultimate aim is to provide adaptive solutions that facilitate a smooth transition between classical and quantum security models, thereby ensuring resilience against the evolving landscape of cyber threats.

Research Lead: Rohit Bantupalli (rohit.b@iiits.in)
